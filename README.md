# Blinkit Sales Performance & Insights Dashboard
## Project Overview
This business intelligence dashboard provides a comprehensive analysis of sales performance and retail operations inspired by the **Blinkit** (Quick-Commerce) ecosystem. By processing transactional data, this project delivers data-driven insights into customer purchasing behavior, outlet distribution performance, and item category revenue splits. The final interactive report helps identify optimized growth strategies for quick-commerce supply chains and dynamic inventory management.
